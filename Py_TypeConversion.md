## `int`

You can convert a value to integer using the `int()` function:

```py
string = type("1")
integer = type(int("1"))

print(string)
# <class 'str'>

print(integer)
# <class 'int'>
```

---

---

## `float`

`float` converts integers and strings to floating-point numbers:

```py
print(float(32))
# 32.0
print(float('3.14159'))
# 3.14159
```

---

---

## `str`

Many Python objects can be converted to a string using the `str` function:

```py
print(type(str(12)))
# <class 'str'>
```
